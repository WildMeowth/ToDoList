# todolist

> 一个简单的Vue.js应用，这个分支是master的vue单文件版本，实现排序,删除,添加item的功能及其过渡效果

## 运行这个应用

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run all tests
npm test
```
